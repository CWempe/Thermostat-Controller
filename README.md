## define a Switch

Not all GPIOs work for switches.
successfully tested:

```
GPIO1
GPIO3
GPIO4
GPIO12
```

Execute via Console:
```
SwitchMode1 2
SwitchTopic1 <device>-switch
```

different than the actiual device/topic name
